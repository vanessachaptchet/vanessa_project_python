# File Organizer in Python

This project is a simple Python automation script that organizes files in a folder based on their file extensions.

## Features

- Creates folders automatically
- Sorts files by type
- Handles:
  - CSV files
  - Images
  - Text documents
  - Excel files
  - Code files
  - Other files

## Technologies Used

- Python
- os
- shutil
